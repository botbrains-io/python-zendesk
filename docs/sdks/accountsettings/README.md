## Overview

### Available Operations

* [show_account_settings](#show_account_settings) - Show Settings
* [update_account_settings](#update_account_settings) - Update Account Settings

## show_account_settings

Shows the settings that are available for the account.

#### Allowed For

* Agents


### Example Usage

```python
from zendesk import Zendesk, models


with Zendesk(
    security=models.Security(
        username="",
        password="",
    ),
) as z_client:

    res = z_client.account_settings.show_account_settings()

    # Handle response
    print(res)

```
