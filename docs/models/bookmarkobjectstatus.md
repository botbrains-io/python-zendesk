# BookmarkObjectStatus

The state of the ticket.

If your account has activated custom ticket statuses, this is the ticket's
status category. See [custom ticket statuses](#custom-ticket-statuses)



## Values

| Name      | Value     |
| --------- | --------- |
| `NEW`     | new       |
| `OPEN`    | open      |
| `PENDING` | pending   |
| `HOLD`    | hold      |
| `SOLVED`  | solved    |
| `CLOSED`  | closed    |
