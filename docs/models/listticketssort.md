# ListTicketsSort

Sort tickets by field (for cursor pagination use "field" for ascending or "-field" for descending)


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `UPDATED_AT`       | updated_at         |
| `ID`               | id                 |
| `STATUS`           | status             |
| `MINUS_UPDATED_AT` | -updated_at        |
| `MINUS_ID`         | -id                |
| `MINUS_STATUS`     | -status            |
